What is HTML and what does it do?

HTML stands for HyperText Markup Language. It is used to create webpages which is pretty cool! It is mainly used to describe the structure of the webpage and HTML elements tell the browser how to display the content.

Basic HTML

<!DOCTYPE html>
      <html>
        <head>
        </head>
        <body>
        </body>
      </html>

HTML Section

The HTML tag is a representation of the root of an HTML document. It also acts as a container for all the other HTML elements except !DOCTYPE.

Head Section

The header tag is a container for the metadata about the HTML document. The metadata is not displayed but it is useful as it contains information about the document title, character set, styles, scripts, and other meta information.

Body Section

This portion of the HTML document contains all the contents of the document such as such as headings, paragraphs, images, hyperlinks, tables, lists, and other things. Also, it is worth mentioning there can only be one body in an HTML document.
